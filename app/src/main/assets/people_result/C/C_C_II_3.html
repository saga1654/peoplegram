두 사람 관계에서 님의 긍정적인 면
1. 님은 부하직원을 세심하게 배려하고 부드러운 분위기를 만드는 능력이 뛰어납니다.
2. 님은 부하직원의 이야기를 잘 들어 줍니다.
3. 님은 자신의 감정을 잘 표현하고 부하직원들과 소통도 매우 잘 합니다.
4. 님은 부하직원과의 대화 속에서 많은 정보와 지식을 습득합니다.
5. 님은 부하직원들의 갈등을 잘 조정하고 협력적인 팀워크를 만드는 데 뛰어납니다.
6. 님은 부하직원의 업무를 잘 지원하고 안정적으로 일을 수행하게끔 하여 부하직원들이 의지하고 신뢰합니다.
Key : 부하직원 님을 지지하고 지원해 주세요.
1. 부하직원 님의 기분이나 감정에 너무 예민하게 신경 쓰지 마세요.
2. 부하직원 님이 어떻게 생각할 지 등에 신경 쓰지 말고 님과 조직의 과제를 빠르게 성공적으로 완료하는데 집중하세요.
3. 일을 할 때는 목표 일정을 정하고 계획을 수립한 후에 실행하세요.
4. 님과 조직이 맡은 일이 문제가 있고 위험할 수 있더라도 용기 있게 시도해 보세요.
5. 님과 부하직원들의 업무 환경을 보다 빠르고 업무 중심적인 환경으로 바꿔 보세요.
6. 부하직원 님의 입장이 되어 내가 일을 대신한다는 입장으로 생각하고 대처해 보세요.
7. 님께서 매일 반복하고 있는 업무(회의, 결재, 승인 등)는 새로운 절차나 방법을 찾아 바꿔 보세요. 보다 능률(효율)적이고 창의적인 방법을 찾아 보세요.
8. 님과 부하직원 님 사이에 갈등이나 문제가 발생되면 님의 감정이나 생각, 경험 등을 부하직원 님과 솔직하게 얘기해 보세요.
